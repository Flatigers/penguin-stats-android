**正在开发中**

A simplified Android app for the [Penguin-Stats]

这是一款 [企鹅物流数据统计] 的简洁安卓客户端

数据来源：

- [企鹅物流数据统计](https://penguin-stats.cn/)
- [Penguin Statistics](https://github.com/penguin-statistics)
